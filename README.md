## Running Tests

To run tests, run the following command

```bash
  npm test
```


## Run Locally

To start electron forge
```bash
  npm start
```

To package electron forge
```bash
  npm package
```

To install depencies (first run)
```bash
  npm install
```

To package electron forge
```bash
  npm make
```

To package electron forge
```bash
  npm package
```
